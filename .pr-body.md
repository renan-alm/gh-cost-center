## PR 1: Go Project Scaffold + CLI Skeleton

First step of the Python to Go conversion for `gh-cost-center`, a precompiled GitHub CLI extension.

### What's included

| File | Purpose |
|------|---------|
| `main.go` | Go entry point |
| `cmd/root.go` | Root command with `--config`, `--verbose` global flags |
| `cmd/assign.go` | `assign` subcommand with all flags (--mode, --teams, --repo, --yes, --incremental, --create-cost-centers, --create-budgets, --check-current, --users) |
| `cmd/list_users.go` | `list-users` subcommand |
| `cmd/config.go` | `config` subcommand (show configuration) |
| `cmd/report.go` | `report` subcommand (--teams) |
| `cmd/cache.go` | `cache` subcommand (--stats, --clear, --cleanup) |
| `cmd/version.go` | `version` subcommand (reads VERSION file, supports -ldflags injection) |
| `Makefile` | build, install, test, lint, clean, fmt, vet, tidy targets |
| `.github/workflows/release.yml` | Cross-compiled binary releases via gh-extension-precompile |
| `.github/workflows/ci.yml` | Go build + test + vet + golangci-lint CI |
| `.gitignore` | Updated with Go build artifacts |
| `go.mod` / `go.sum` | Go module with cobra, go-gh/v2, yaml.v3, fatih/color |

### Usage after install

```bash
gh extension install .
gh cost-center --help
gh cost-center assign --help
gh cost-center version
```
